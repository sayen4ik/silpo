Гра-прототип

Створила невеликий баланс-гейм: натискаєш стрілки на екрані чи клавіатурі — утримуєш рівновагу.”

Анімаційні стани:

  • Балансуючий аспарагус
  • Завалюється праворуч
  • Завалюється ліворуч

(Повернення до балансу через реверс анімації.)

Це лише базові Lottie JSON анімації, які в подальшому слід деталізувати для кращої емоційної виразності.

Можливе розширення концепту:

  • Ускладнення фізики (вітер, рух платформи, падаючі об’єкти)
  • Додавання ефектів (цифровий шум, звукові ефекти)
  • UI/UX — показати прогрес, таймер, стан героя тощо.
